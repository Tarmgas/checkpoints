# checkpoints
checkpoints for Dyngecoin

Tested so far:

LINUX

just unpack, copy to dyngecoin-folder and run:

./Dynged --load-checkpoints checkpoints.csv

WINDOWS

just unpack, copy to dyngecoin-folder and run:

dynged --load-checkpoints checkpoints.csv

in windows GUI, just unpack, copy to gui-folder and run with local blockchain and checkpoints checked.
